# HP Pavillion Gaming Laptop 15-cx0xxx (aka. 光影精靈四代) Hackintosh OpenCore EFI

### Working
- Intel UHD 630 Graphics Card
- Elan Touchpad
- Realtek ALC295 Speakers
- Headphone Jack
- Realtek Ethernet Adapter
- USB 3.1
- Intel AC9560 Wireless Card
- Hibernation
